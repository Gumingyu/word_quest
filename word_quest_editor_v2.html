使用说明

🎯 这个工具是干嘛的？

让你完全脱离代码，自己管理 Word Quest 游戏的题库。所有改动都保存在浏览器里，导出 JSON 后可以分享给学生或备份。

✏️ 基本操作

  • 左侧选择关卡（5个章节关卡）
  • 点击题目卡片可编辑，悬停显示删除按钮
  • 右上"+ 添加题目"创建新题
  • 右下"实时预览"看学生端的呈现效果

🎨 三种题型设计

  • 基础攻击 (10 伤害)：单词中英互译，最简单
  • 暴击 (25 伤害)：词组搭配、反义词、语境填词
  • 必杀 (50 伤害)：语法选择、句子翻译，最难

📥 导入 / 📤 导出

  • 导出 JSON：保存当前题库到本地文件，作为备份或分享
  • 导入题库：上传之前导出的 JSON 文件，恢复或替换题库
  • 复制到游戏：生成可以粘贴到游戏 HTML 文件里替换题库的代码

💡 出题小贴士

  • 题干用 ___ 表示填空（3个下划线）
  • 音标用斜杠包裹：/ˈwɔːtər/
  • 例句最好和题目主题相关，便于课堂引申
  • 干扰项要"像但不对"——用学生真实犯的错来做干扰

⚠️ 注意

题库改动只保存在你当前浏览器里。关闭页面前记得导出 JSON，否则换电脑就找不到了。
